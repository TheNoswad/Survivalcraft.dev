# Survivalcraft API
A community effort has led to the development of the "Survivalcraft API" client. This version of the game is a close fork of the vanilla game, but implements plug-in modules in the form of `.DLL` files. 

## Features
- Plug-in modules
- Maximum visibility range increased to 2048 blocks.
- Field of view configuration option
- Multilingual Support
- Additional community content repositories
- Window mode options
- Advanced memory bank editing interface
- Keyboard remapping
- Gamepad remapping
- Debug Camera
- Support for much larger skins and textures

## Obtaining the client
